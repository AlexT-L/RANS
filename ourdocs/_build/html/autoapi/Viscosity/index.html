Viscosity.compute_viscosity(model, ws, state)¶
Computes viscosity coefficients.

First, computes the molecular viscosity. Then continues for turbulent, Baldwin Lomax Model or runs the RNG algebraic model. Next, calculates the boundary layer thickness Solves for the eddy viscosity.
