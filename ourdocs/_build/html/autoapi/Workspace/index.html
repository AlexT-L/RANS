init_vars(self, className, vars)¶

Initialize class’s stored fields, must give a dictionary of variables with the following structure: - keys are the variable name as a string (e.g. “w”) - values are an array of [field_dimensions, state_dim]

  • field_dim is usually the grid size in [nx, ny] format

  • state_dim is 1 for scalars
