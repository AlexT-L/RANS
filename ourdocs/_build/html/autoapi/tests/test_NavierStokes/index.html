tests.test_NavierStokes¶

Tests the NavierStokes object to make sure it behaves as expected

Libraries/Modules:

pytest Input NavierStokes AirfoilMap CellCenterWS NS_Airfoil

Notes

Runs the following tests: 1. Checks that the constructor works 2. Checks that the airfoil geomtry is of the right length

  1. Check that all the required dims values are being read into the dims dictionary

  2. Check that all the required solv_param values are being read into the solv_param dictionary

  3. Check that all the required flo_param values are being read into the flo_param dictionary

  4. Check that all the required geo_param values are being read into the geo_param dictionary
