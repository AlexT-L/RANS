tests.test_Workspace¶

Description

Tests the Workspace object to see if it works as expected

Libraries/Modules

-pytest

-Workspace

Notes

Runs the following tests:

Verify x, xc, vol can be retrieved Verify x, xc, vol are non-zero Verify that init_vars works as expected then check that init_vars? exists return True check that has_dict? exists and has_dict return false when they should check is_finest()
